# Анализ посещений сайта – Декабрь 2024

## Описание задания
Файл `website_visits_december_2024.csv` содержит данные о поведении пользователей сайта за декабрь 2024 года:

- **Дата посещения** – DD-MM-YYYY  
- **День недели** – пн, вт, …  
- **Время посещения** – HH:MM  
- **Тип устройства** – Мобильное / Десктоп  
- **Просмотренные страницы** – количество страниц за визит  
- **Время на сайте (сек)** – продолжительность визита  

Задача: создать функцию `website_analyse`, которая:

1. Вычисляет среднее количество просмотренных страниц  
2. Рассчитывает стандартное отклонение времени на сайте  
3. Рассчитывает коэффициент вариации и выводит категорию вариабельности:  
   - >0.25 → Высокая  
   - 0.1–0.25 → Умеренная  
   - <0.1 → Слабая  
4. Проверяет статистически значимую разницу между пользователями Мобильных и Десктопных устройств с помощью t-теста  

Все результаты выводятся через `print()`.  

## Используемые библиотеки
- `pandas` – работа с CSV  
- `scipy.stats` – t-тест
